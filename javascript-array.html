JavaScript 陣列 Q&A

陣列的宣告、讀取、常用方法與實際應用。

Q1：陣列是什麼？

陣列是一個可以「放很多資料」的變數，資料有順序，而且可以用數字來存取。

let fruits = ["apple", "banana", "kiwi"];
Q2：陣列要怎麼宣告？
// 最常用寫法
let numbers = [10, 20, 30];

// 也可以先建空陣列
let data = [];
data.push(100);
data.push(200);
Q3：怎麼取出第 n 個資料？

陣列是從 第 0 個開始算！

let colors = ["red", "green", "blue"];

console.log(colors[0]); // red
console.log(colors[1]); // green
console.log(colors[2]); // blue
Q4：怎麼知道陣列有幾個值？

用 .length

let food = ["rice", "noodle", "dumpling"];

console.log(food.length); // 3
Q5：陣列常用的「新增 / 刪除 / 搜尋」方法有哪些？
let arr = [1, 2, 3];

// 加在最後
arr.push(4);     // [1,2,3,4]

// 移除最後
arr.pop();       // [1,2,3]

// 加在最前
arr.unshift(0);  // [0,1,2,3]

// 移除最前
arr.shift();     // [1,2,3]

// 搜尋 index
arr.indexOf(2);  // 回傳 1
Q6：怎麼快速處理全部陣列內容？

可以用 forEach()

let scores = [70, 85, 90];

scores.forEach(function(score) {
  console.log("分數：" + score);
});
或用 map() 回傳新陣列： 
let double = scores.map(n => n * 2);
console.log(double); // [140, 170, 180]
Q7：實際應用：計算陣列平均分數怎麼寫？
let scores = [80, 90, 100, 70, 60];

let sum = 0;
scores.forEach(s => sum += s);

let avg = sum / scores.length;

console.log("平均分數：" + avg);
