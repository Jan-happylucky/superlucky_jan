色彩與字形介紹 Q&A

配色與文字樣式的小技巧。

Q1：怎麼選一組安全又好看的字型？

常見的做法會寫一串「字型 fallback」：

body {
  font-family: system-ui, -apple-system, BlinkMacSystemFont,
               "Segoe UI", sans-serif, "Microsoft JhengHei";
}

意思是：如果前面沒有，就用後面的；確保在 Windows / Mac / 手機看起來都還算正常。

Q2：文字顏色要怎麼選才不會太刺眼？
  • 不要用純黑 #000000 + 純白 #ffffff，對比太強。
  • 可以改用稍微軟一點的顏色，例如字用 #1f2933 或 #374151。
這樣看久比較不累。
Q3：標題跟內文要怎麼區分層級？

可以同時調整：

  • 字體大小（font-size）
  • 字重（font-weight）
  • 顏色深淺（標題深一點、內文淡一點）
例如： 
h1 {
  font-size: 28px;
  font-weight: 700;
}
p {
  font-size: 14px;
  color: #6b7280;
}
