九宮格示範 Q&A

說明首頁的 RWD 九宮格是怎麼做出來的。

Q1：首頁的九宮格為什麼在手機上會變成一欄？

首頁有一段 CSS 大概是這樣：

.grid {
  display: grid;
  grid-template-columns: 1fr;
  gap: 14px;
}

@media (min-width: 640px) {
  .grid {
    grid-template-columns: repeat(2, minmax(0, 1fr));
  }
}

@media (min-width: 960px) {
  .grid {
    grid-template-columns: repeat(3, minmax(0, 1fr));
  }
}

意思是：

  • 小螢幕（小於 640px）：1 欄
  • 中螢幕（大於等於 640px）：2 欄
  • 大螢幕（大於等於 960px）：3 欄
Q2：RWD 的重點是什麼？
  • 不要死抓固定寬度（例如一直用 width: 1200px;）。
  • 多用百分比、max-width、grid、flex 等彈性排版。
  • 用 @media 搭配不同的螢幕寬度調整版型。
