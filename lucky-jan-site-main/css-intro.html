CSS 語法介紹 Q&A

selector、屬性、值的基本觀念。

Q1：CSS 的基本句子長什麼樣子？
選擇器 selector {
  屬性: 值;
}
例如： 
p {
  color: red;
  font-size: 16px;
}
意思是：把所有的 <p> 文字改成紅色、字型大小 16px。
Q2：class 和 id 差在哪？
  • class：同一個 class 可以用在很多元素上，像貼同一種標籤。
  • id：一個網頁中通常只會有一個一樣的 id，比較像身分證字號。
CSS 選取方式：
  • .box { ... }：選到 class="box" 的元素。
  • #header { ... }：選到 id="header" 的元素。
Q3：行內樣式、內部樣式、外部樣式有什麼差別？
  • 行內樣式：寫在標籤裡面的 style=""。
    <p style="color: red;">文字</p>
  • 內部樣式：寫在 <style> 標籤裡（通常放在 head）。
  • 外部樣式：獨立一個 .css 檔，再用 <link> 引入。
實務上會偏好「外部樣式」，因為比較好維護。
