常用 CSS 語法推薦 Q&A

顏色、字型、間距、圓角、陰影等實用屬性。

Q1：如何設定文字的顏色、大小、行距？
p {
  color: #333333;
  font-size: 16px;
  line-height: 1.6;
}
  • color：文字顏色
  • font-size：文字大小
  • line-height：行高（1.5~1.8 通常比較好讀）
Q2：margin 跟 padding 有什麼差？

可以把一個盒子想像成禮物盒：

  • padding：內容跟盒子邊框之間的「內距」，像是保護的泡棉。
  • margin：盒子跟盒子之間的「外距」，決定盒子之間要不要離遠一點。
Q3：怎麼讓按鈕有一點圓角跟陰影？
.btn {
  background: #4c6fff;
  color: #ffffff;
  padding: 8px 16px;
  border-radius: 999px;
  border: none;
  box-shadow: 0 6px 12px rgba(76,111,255,0.35);
}
  • border-radius：控制圓角（999px 會變成膠囊形狀）。
  • box-shadow：增加立體感。
