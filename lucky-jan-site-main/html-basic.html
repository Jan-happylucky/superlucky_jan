HTML 基本架構 Q&A

關於 <html>、<head>、<body> 的常見問題與解答。

Q1：一份最基本的 HTML 網頁一定要有哪些標籤？
一般來說，最基本的 HTML 架構會包含這幾個部分： 
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>我的第一個網頁</title>
</head>
<body>
  這裡是網頁的內容
</body>
</html>

重點：
  • <!DOCTYPE html>：告訴瀏覽器這是 HTML5。
  • <head>：放網頁的「設定、資訊」，不是可見內容。
  • <body>：使用者真正看到的內容都放這裡。
Q2：為什麼要加 lang="zh-Hant" 跟 <meta charset="UTF-8">？

lang="zh-Hant" 是告訴瀏覽器「這個網頁主要是繁體中文」，有助於：

  • 搜尋引擎理解內容語言
  • 螢幕報讀器（給視障者用的）正確發音

<meta charset="UTF-8"> 是設定「文字編碼」，如果沒設好，中文可能會變亂碼。

Q3：<head> 跟 <body> 差在哪裡？

可以把一個網頁想像成一個「人」：

  • head：像是大腦，放的是設定、標題、關鍵字、引用的 CSS、JS 等，看不到但很重要。
  • body：像是身體，放的是文字、圖片、按鈕、表格等等，使用者實際看到的東西。
Q4：一個網頁可以有兩個 <body> 嗎？
不行，標準的 HTML 文件結構中：
  • <html> 只能有一個
  • <head> 只能有一個
  • <body> 也只能有一個
如果寫了兩個 <body>，瀏覽器有時候會「幫你自動修正」，但那不是正確寫法，考試或作業會被扣分喔。
