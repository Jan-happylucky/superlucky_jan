版型與排版介紹 Q&A

Flex、Grid、版心與留白的基本觀念。

Q1：什麼是「版心」？

版心可以理解成「中間那塊主要內容的區域」。

常見做法是給一個容器：

.container {
  max-width: 1200px;
  margin: 0 auto;
}
這樣畫面再寬，內容也不會拉太長，閱讀起來比較舒服。
Q2：Flex 主要用來做什麼？

Flex 很適合用來做「一排」的排版，例如：

  • 導覽列（logo + 選單）
  • 三欄或四欄卡片
.row {
  display: flex;
  gap: 16px;
}
只要在父層加上 display: flex;，裡面的子元素就會自然排成一排。
Q3：Grid 跟 Flex 有什麼不一樣？
  • Flex：偏重「一個方向」的排列（橫排或直排）。
  • Grid：偏重「二維」的網格，適合九宮格、棋盤狀排版。 
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
}
這個就是典型九宮格的設定。
